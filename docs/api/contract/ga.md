Provide Generalize Account implementation

**Kind**: Exported function  
**Returns**: `Object` - GeneralizeAccount instance  
**rtype**: `Stamp`

| Param | Type | Default | Description |
| --- | --- | --- | --- |
| [options] | `Object` | <code>{}</code> | Initializer object |

**Example**  
```js
const authContract = ``
```
