Class AeSdkMethods

AeSdkMethods is the composition of:

  • chain methods
  • tx methods
  • aens methods
  • spend methods
  • oracle methods
  • contract methods
  • contract ga methods

While these methods can be used separately, this class provides a handy way to store their context (current account, network, and compiler to use).
